# covid19_console
A small python3 CLI program for Linux that outputs the COVID-19 stats based on your locality (India only) 

## Dependences
wget

## Usage
Run the program with `python3 covid.py`.
Config files and json are stored in `/tmp/pycovid/`.

## Sources:
1. [The covid19india.org website](https://api.covid19india.org/)
2. [COVID-19 REST API for India](https://api.rootnet.in/)
